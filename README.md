## Tech stack :
### Frontend :
- React
- Redux
- Tailwind CSS
- REST API
- Responsiveness for all screen size



## Run Locally :
Clone the Project
```
https://github.com/owii8247/Md_Owais_Athar-Frontend_Developer.git
```

```
npm install
```

```
npm run start
```
